## What is this?
A garage where I build, test, and break machine learning models on F1 data.

No podiums. No constructors' championships. Just a rookie driver in the ML paddock learning the track lap by lap.

## How to run this garage?

git clone https://github.com/yourusername/f1-ml-lab

cd f1-ml-lab/lap-time-predictor

pip install -r requirements.txt

streamlit run lap-time-predictor/app.py


Or try the live app! https://f1-ml-lab.streamlit.app/

## Tech Stack

- Python
- FastF1 (data)
- Scikit-learn
- Streamlit (UI)
- Google Colab (training)
